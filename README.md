# 新聞配達マップ共有

Google Maps上で新聞配達の配達予定、未配、誤配、配達ルートを管理する静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 設定タブで Google Maps APIキーを入力して地図を読み込みます。
3. 名簿タブに `名前,住所,エリア,部数,メモ` 形式で名簿を貼り付けます。
4. 名簿候補を地図上へドラッグするか、候補を選択して地図をクリックするとピンを登録できます。
5. 登録済みピンはドラッグして位置を修正できます。
6. ピンをクリックすると、完了・未配・誤配を登録できます。

## スマホで使う場合

GitHub Pagesなどの静的ホスティングに公開するとスマホからアクセスできます。

Google Maps APIキーは、公開URLだけで使えるようにHTTPリファラー制限を設定してください。

例:

```text
https://ob198-cpu.github.io/sinbun/*
```

必要なAPI:

- Maps JavaScript API
- Geocoding API
